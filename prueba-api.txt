Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde con un objeto JSON que confirma que los datos fueron recibidos correctamente. La respuesta es:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que el servidor está utilizando correctamente req.body para capturar la información enviada por el cliente.
